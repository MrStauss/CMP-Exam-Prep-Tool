# CMP Study Tool - Interaction Design

## Core Features

### 1. Domain-Based Study System
- **Domain Dashboard**: Visual overview of all 12 CMP domains with progress tracking
- **Weighted Study Priority**: Color-coded system showing exam weight (Event Design 25% = highest priority)
- **Progress Tracking**: Track mastery level for each domain and sub-domain

### 2. Interactive Flashcards
- **Key Terminology**: 20+ essential CMP terms with definitions
- **Domain-Specific Cards**: Organized by the 12 exam domains
- **Study Modes**: 
  - Flip cards (term â†’ definition)
  - Quiz mode (multiple choice)
  - Self-assessment (rate your knowledge)
- **Progress Tracking**: Mark cards as "Mastered", "Needs Review", "Learning"

### 3. Practice Questions & Scenarios
- **Domain-Specific Questions**: 10+ questions per domain (150+ total)
- **Multiple Choice Format**: Mimics actual exam structure
- **Scenario-Based Questions**: Real-world event planning situations
- **Instant Feedback**: Immediate explanations for correct/incorrect answers
- **Weak Area Identification**: System tracks which domains need more study

### 4. Study Progress Dashboard
- **Visual Progress Bars**: Show completion percentage for each domain
- **Study Time Tracking**: Log hours spent on each domain
- **Performance Analytics**: Success rates on practice questions
- **Study Recommendations**: AI-powered suggestions for focus areas

### 5. Exam Readiness Assessment
- **Simulated Exam Mode**: 150 questions matching actual exam distribution
- **Timed Practice**: Simulate real exam conditions
- **Score Tracking**: Monitor improvement over time
- **Domain Breakdown**: Detailed performance analysis by domain

## User Interaction Flow

1. **Dashboard Entry**: User sees overview of all domains with progress indicators
2. **Domain Selection**: Click on any domain to access specific study materials
3. **Study Mode Selection**: Choose flashcards, practice questions, or scenarios
4. **Progress Tracking**: System automatically tracks and visualizes learning progress
5. **Adaptive Learning**: Tool recommends next study areas based on performance

## Interactive Components

### Component 1: Domain Progress Tracker
- Visual circular progress indicators for each domain
- Click to drill down into specific skills and sub-skills
- Color-coded priority system based on exam weight

### Component 2: Adaptive Flashcard System
- Flip animation for term/definition reveal
- Self-assessment buttons (Know It, Need Review, Don't Know)
- Shuffle and filter by domain or mastery level

### Component 3: Practice Question Engine
- Multiple choice with immediate feedback
- Scenario-based questions with detailed explanations
- Performance tracking and weak area identification

### Component 4: Study Analytics Dashboard
- Charts showing study time distribution
- Performance trends over time
- Personalized study recommendations
